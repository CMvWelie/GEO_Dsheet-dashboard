Waar komt het Word-preview paneel?

Kies de plek van het preview-venster in de applicatie

A

Uitschuifbaar paneel rechts (splitter)

Preview schuift in/uit aan de rechterkant van het hoofdvenster via een knop of knop in de toolbar. Blijft binnen hetzelfde applicatievenster.

D-Sheet Dashboard
Tabs (Import / Invoer / Export…)
Tabinhoud
Word Preview
Rapport titel
1. Invoer...
2. Resultaten...
B

Apart zwevend venster

Preview opent als een apart Qt-venster. Je kan het los naast of boven het hoofdprogramma plaatsen, desgewenst op een tweede monitor.

D-Sheet Dashboard
Hoofdapplicatie...
Word Preview ↗
Rapport
1. Invoer
2. Resultaten
C

Extra tabblad "Preview"

Preview is gewoon een extra tab in de bestaande tabstructuur. Eenvoudigst te bouwen, maar je ziet de preview niet naast andere inhoud.

D-Sheet Dashboard
Import Invoer Word Preview ★ Export
Rapport — Live preview
1. Invoerbeschrijving...
2. Resultaten...
