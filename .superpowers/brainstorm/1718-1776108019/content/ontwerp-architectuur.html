Dataflow
App start
→
ConfigManager laadt AppSettings → AppState.app_settings
Template-pad wijzigt
→
template_path_changed signal → AppController.apply_app_settings() → state + config opslaan
_update_all()
→
Als preview open → HtmlPreviewBuilder.build(package) → preview_window.set_html(html)
Word-export
→
Gebruikt AppState.app_settings.word_template_path als fallback (als Word-tab leeg is)
Ontwerpbeslissing: WordPreviewWindow heeft geen directe toegang tot AppState. Hij ontvangt alleen een HTML-string via set_html(). Dit houdt het venster dom (geen logica) en makkelijk te testen.
